Installation
------------

The preferred way to install this extension is through [composer](http://getcomposer.org/download/).

Either run

```
php composer.phar require --prefer-dist yii2tech/ar-search
```

or add

```json
"yii2tech/ar-search": "*"
```

to the require section of your composer.json.
